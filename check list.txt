Идея
Сделать личного финаносвого помощника, позволящий хранить данные о расходах и доходах,
присылать аналитику расходов,
выводить информацию о текущих курсах валют, подсказывать о высоких расходах и
 давать рекомендации по снижению.
 
 Реализация
 В качестве интерфейса работы с пользователем используются телеграм бот. 
 
   при отправке сообщения /start - регистрация пользователя
							(1 команда на выбор)
   если отправить сообщение [{!s !spent /s /spent} {сумма расходов} {информация о расходах}],
   бот сохранит данные в БД
							(1 команда на выбор)
   если отправить сообщение [{!e !earned /e /earned} {сумма доходов} {информация о доходах}],
   бот сохранит данные в БД
							(1 команда на выбор)
   если отправить сообщение [{!h !history /h /history} {варинат истории}], выводит историю 
   расходов/доходов. 
   Варианты истории(если ниче не вводить, по умолчанию будет статистика за месяц)
   {
	today, day, сегодня, день - за день
	month, месяц - за текущий месяц
	year, год - за год
   }
   
 
   если отправить сообщение [{!courses !курсы /courses /курсы} {имя банка}]
   Имя банка (пустое поле выводит все курсы)
   {
    bsb, бсб - БСБ банк
	alfa, альфа - альфа банк
	belweb, белвэб - бел вэб
	prior, приор - приорбанк
   }
	
   ############
   В разработке
   ############
   если отправить сообщение [{!stash !заначка /stash /заначка}]
   Если нету пользователя в базе предложит создать пароль для доступа,
   если есть попросит ввести пароль для доступа к заначке,
   при успешном вводе выведет значение(сумму)
   выход из заначки команда {!out /out !выход /выход}
